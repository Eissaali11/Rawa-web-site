## 📌 نظرة عامة

**Rawa Workforce** هو موقع ويب تعريفي احترافي لشركة **رايا** المتخصصة في خدمات توظيف القوى العاملة في المملكة العربية السعودية. يتميز بتصميم عصري متجاوب مع لوحة تحكم إدارية مدمجة.

```
╔══════════════════════════════════════════════════════╗
║           🌟 RAWA WORKFORCE WEBSITE                  ║
║                                                      ║
║  ┌──────────────┐  ┌──────────────┐  ┌───────────┐  ║
║  │  🏠 الرئيسية │  │  📋 الخدمات  │  │ 📞 تواصل  │  ║
║  └──────────────┘  └──────────────┘  └───────────┘  ║
║                                                      ║
║  ┌──────────────────┐  ┌────────────────────────┐   ║
║  │  🔐 تسجيل الدخول │→ │  📊 لوحة التحكم        │   ║
║  └──────────────────┘  └────────────────────────┘   ║
╚══════════════════════════════════════════════════════╝
```

## ✨ المميزات الرئيسية

| الميزة | الوصف |
|--------|-------|
| 📱 **تصميم متجاوب** | يعمل على جميع الأجهزة (جوال، تابلت، كمبيوتر) |
| 🔐 **نظام مصادقة** | لوحة تحكم إدارية محمية |
| 📊 **لوحة التحكم** | إدارة مركزية للبيانات |
| 🎨 **تصميم احترافي** | واجهة جذابة بألوان الشركة |
| 🤖 **SEO محسّن** | Robots.txt وSitemap.xml |
| ⚡ **أداء عالي** | Vanilla JS خفيف وسريع |

## 🚀 طريقة التشغيل

```bash
# 1. نسخ المستودع
git clone https://github.com/Eissaali11/Rawa-web-site.git
cd Rawa-web-site

# 2. فتح الموقع (طريقة سريعة)
open index.html

# أو باستخدام سيرفر محلي
python -m http.server 8000
# http://localhost:8000
```
